# ZX81 Emulator

### 1.0.0 (August 1, 2017)

* Works in Chrome, Firefox, Safari, Edge but not in IE
* Is does **not** currently work on Internet Explorer
* Runs the game `FACTORY` I made in 1985 (translated from french)
